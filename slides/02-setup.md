# Enabling Agent Mode

1. **Settings**: Turn on `chat.agent.enabled` in VS Code settings
2. **Access**: Open Copilot Chat panel
3. **Switch**: Change chat mode to **Agent** (instead of Ask/Edit)
4. **Request**: Type your task and approve when prompted
